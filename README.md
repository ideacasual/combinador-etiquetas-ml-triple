# Optimizador de Etiquetas de Envío (A4)

Herramienta web gratuita para combinar hasta 3 etiquetas de Mercado Libre en una sola hoja A4, optimizando el uso de papel y tinta al momento de imprimir.

**[Probar la herramienta en vivo aquí](https://etiquetas.ideacasual.com/)**

---

### ¿Por qué existe esta herramienta?

Las etiquetas de Mercado Libre (Argentina) suelen generarse con 2 páginas:

1. **Página 1:** La etiqueta de envío propiamente dicha (formato apaisado).
2. **Página 2:** La información del producto (formato vertical).

**El problema:** Imprimir la página 2 desperdicia papel, pero descartarla hace muy difícil identificar a qué producto corresponde cada etiqueta si no se empaca inmediatamente.

**La solución:** Este programa fusiona ambas páginas en una sola hoja A4. La etiqueta queda lista para imprimir y pegar, con el pequeño dato del producto ubicado estratégicamente en la misma hoja para ser recortado fácilmente.

Eso nos permite agregar hasta tres etiquetas con sus datos extra en una misma hoja A4 apaisada.

---

### Cómo usarla

1. **Subí tus archivos:** Arrastrá o seleccioná tus etiquetas (máximo 10 archivos por sesión).
2. **Requisitos:** Los archivos deben ser `.pdf` de exactamente 2 páginas (Página 1 apaisada, Página 2 vertical).
3. **Elegí el modo de procesamiento:**
   - **Conjuntos:** Agrupa hasta 3 etiquetas optimizadas en una sola hoja A4 (ideal para ahorro máximo de papel).
   - **Individuales:** Una etiqueta optimizada por cada hoja A4.
4. **Descargá:** Obtené tus PDFs combinados listos para imprimir, ya sea de forma individual o comprimidos en un único archivo `.zip`.

---

### Privacidad y Seguridad

- **100% Local:** Todo el procesamiento ocurre directamente en tu navegador.
- **Cero subidas:** Tus archivos **nunca** se envían a ningún servidor, nube o tercero.
- **No destructivo:** Tus archivos originales permanecen intactos; la herramienta solo genera copias nuevas optimizadas.
- **Transparencia total:** Todo el código es visible y ejecutado en tu propio equipo, sin operaciones ocultas.
